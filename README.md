# 🎁 Telegram-бот для организации подарков

Умный и удобный Telegram-бот, разработанный на Java, который помогает пользователю управлять идеями подарков, устанавливать бюджеты для получателей и получать напоминания о предстоящих событиях.

---

## 📌 Возможности

### 🎁 Управление подарками
- Добавление подарков в формате: `Имя - Подарок - Сумма - Дата - Комментарий`
- Редактирование подарков через inline-кнопки (имя, цена, дата, комментарий)
- Удаление отдельных подарков с подтверждением
- Автоматическое обновление статуса на `Завершено` после наступления даты события

### 💸 Бюджеты
- Установка общего бюджета по умолчанию
- Установка индивидуального бюджета для каждого получателя
- Предупреждение при превышении бюджета

### ⏰ Напоминания
- Настройка количества дней до события для напоминания
- Выбор точного времени отправки напоминания
- Ежедневные напоминания пользователю о приближающихся событиях

### 🔎 Фильтрация и поиск
- Фильтрация подарков по:
  - Дате: `до`, `после`, `сегодня`
  - Статусу: `Завершено`, `В ожидании`
- *(Фильтрация по цене была убрана по решению в финальной версии)*

### 🖱️ Удобный интерфейс
- Все действия производятся через кнопки (inline и reply)
- Нет необходимости запоминать команды

### 🗃️ Хранение данных (шаблон DAO)
- Постоянное хранение в JSON-файле
- Временное хранение в памяти (режим тестирования)
- Переключение через `DaoFactory`
