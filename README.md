#  Gift Organizer Bot (Чат-бот в Telegram)

Telegram-бот, помогающий организовать список подарков, отслеживать расходы и получать напоминания о важных событиях.

##  Возможности

-  Добавление подарков по шаблону
-  Просмотр всех подарков по именам
-  Установка бюджета на каждого получателя
-  Установка напоминаний за X дней
-  Редактирование и удаление подарков
-  Сохранение данных в JSON (по ID пользователя)

##  Безопасность

Файл config.properties добавлен в .gitignore и не загружается на GitHub

Персональные данные пользователей хранятся локально в data/, также исключены из репозитория
